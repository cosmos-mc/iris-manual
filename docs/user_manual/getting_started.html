Getting Started¶

본 튜토리얼 문서에서는 IRIS 빅데이터 솔루션(이하 IRIS)를 처음 사용하는 사용자가 IRIS SaaS 기본 스토리지 MinIO에 데이터를 업로드 하는 방법을 설명합니다. (IRIS는 클라우드 스토리지 서비스인 MinIO 를 기본 스토리지로 제공합니다. MinIO는 아마존 S3와 호환되는 클라우드 스토리지 서비스이며, 자세한 사항은 MinIO 공식 홈페이지 에서 확인하실 수 있습니다)

대상 데이터는 접근성 및 데이터의 품질을 고려하여 서울시에서 제공하는 공공 데이터를 선정하였습니다.
