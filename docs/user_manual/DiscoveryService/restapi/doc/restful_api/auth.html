5.4.1. Auth API¶

5.4.1.1. 개요¶

  • IRIS Discovery Service 의 API 를 사용하기 위해 ID, Password 를 이용하여 토큰을 발행하기 위한 API

5.4.1.2. Generate token¶

5.4.1.2.1. URL¶

  • /angora/auth

5.4.1.2.2. Method¶

  • POST

5.4.1.2.3. Description¶

  • 토큰을 발급 받는 API

5.4.1.2.4. Parameters¶

5.4.1.2.5. Example¶

  • Request
curl -XPOST "http://localhost:6036/angora/auth" -H "Content-Type: application/json" -d '{
    "id" : "root",
    "password" : "password",
    "group_id" : "groupA"
}'
  • Success Response
{
    "status" : "OK",
    "token" : "cm9vdA=="
}
  • Error Response
{
    "status" : "ERROR",
    "token" : null
}
  • Exception
{
    "type": "RuntimeError",
    "message": "This was failed because..."
}
